🎓 Student Performance Prediction System

An AI-based Student Performance Prediction System developed using Flask and Machine Learning.

📌 Features
- Predicts student performance
- Uses Machine Learning model
- User-friendly interface
- Fast prediction
- Web application using Flask

🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- Machine Learning
- Pickle

📁 Project Structure
```
Student_Project/
│── app.py
│── model.py
│── train_model.py
│── model.pkl
│── templates/
│    └── index.html
│── static/
│── student_data.csv
```

⚙️ Installation

1. Clone the repository.
2. Install Python.
3. Install Flask.
4. Run `app.py`.
5. Open the localhost link in your browser.

🚀 How It Works

- Enter student details.
- The model processes the data.
- Predicted performance is displayed instantly.

📊 Output

Displays the predicted student performance.
